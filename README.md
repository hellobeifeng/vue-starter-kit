# vue-starter-kit
一个用于快速搭建前端vue工程的脚手架

### 目前支持功能如下

- [x] 使用自定义 webpack 配置文件 ，灵活性更高:sparkles:
- [x] 支持区分开发模式 和 构建模式
- [x] 支持 eslint 校验
- [x] 支持 precommit git hook
- [x] 使用 webpack-dev-server 运行 开发环境时，自动打开浏览器
- [x] 预设了对 css / jsx / vue / stylus 文件和主流图片类型的文件编译支持
